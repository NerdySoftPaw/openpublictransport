# RVV (Regensburger Verkehrsverbund)

RVV is the transit authority for the Regensburg region in Bavaria, Germany.

## Coverage Area

- Regensburg
- Regensburg metropolitan area

## API Details

| Property | Value |
|----------|-------|
| **Base URL** | `https://efa.rvv.de/efa/` |
| **API Type** | EFA |
| **API Key** | Not required |
| **Timezone** | Europe/Berlin |

## Transport Types

| Class | Type | Description |
|-------|------|-------------|
| 0 | train | High-speed trains (ICE, IC, EC) |
| 1 | train | Regional trains (RE, RB) |
| 5-8 | bus | Bus services |
| 13 | train | Regional (RE) |

## Setup Steps

1. Select **RVV** as provider
2. Search for your stop (e.g., "Hauptbahnhof")
3. Select the stop from the list
4. Configure departure count and filters
