# DING (Donau-Iller-Nahverkehrsverbund)

DING is the transit authority for the Donau-Iller region around Ulm, Germany.

## Coverage Area

- Ulm
- Neu-Ulm
- Donau-Iller region

## API Details

| Property | Value |
|----------|-------|
| **Base URL** | `https://www.ding.eu/ding3/` |
| **API Type** | EFA |
| **API Key** | Not required |
| **Timezone** | Europe/Berlin |

## Transport Types

| Class | Type | Description |
|-------|------|-------------|
| 0 | train | High-speed trains (ICE, IC, EC) |
| 1 | train | Regional trains (RE, RB, S-Bahn) |
| 4 | tram | Tram |
| 5-8 | bus | Bus services |
| 13 | train | Regional (RE) |

## Setup Steps

1. Select **DING** as provider
2. Search for your stop (e.g., "Hauptbahnhof")
3. Select the stop from the list
4. Configure departure count and filters
