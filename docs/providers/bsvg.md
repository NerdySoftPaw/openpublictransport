# BSVG (Braunschweiger Verkehrs-GmbH)

BSVG is the transit operator for Braunschweig in Lower Saxony, Germany.

## Coverage Area

- Braunschweig
- Braunschweig metropolitan area

## API Details

| Property | Value |
|----------|-------|
| **Base URL** | `https://bsvg.efa.de/bsvagstd/` |
| **API Type** | EFA |
| **API Key** | Not required |
| **Timezone** | Europe/Berlin |

## Transport Types

| Class | Type | Description |
|-------|------|-------------|
| 0 | train | High-speed trains (ICE, IC, EC) |
| 1 | train | Regional trains (RE, RB) |
| 4 | tram | Tram |
| 5-8 | bus | Bus services |
| 13 | train | Regional (RE) |

## Setup Steps

1. Select **BSVG** as provider
2. Search for your stop (e.g., "Rathaus")
3. Select the stop from the list
4. Configure departure count and filters
